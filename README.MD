```
Executing LEX Programs:
$ lex fileName.l
$ cc -o out lex.yy.c -ll
$ ./out

Executing LEX and YACC together:
$ cd toFolder
$ yacc -d fileName.y
$ lex fileName.l
$ cc -o out y.tab.c lex.yy.c -ly -ll
$ ./out
```
